# Green Time Tracker - Public Documentation

This repository contains public-facing legal documents for the Green Time Tracker iOS application.

## Documents

- **[Privacy Policy](https://coryroyce.github.io/green-time-tracker-docs/privacy-policy.html)** - Comprehensive privacy policy for the Green Time Tracker app

## About Green Time Tracker

Green Time Tracker is a mobile application designed to help users track and monitor their outdoor activities and time spent in nature.

## Repository Purpose

This repository is maintained separately from the main application codebase to:
- Keep the application source code private
- Make legal documents publicly accessible
- Meet App Store requirements for privacy policy hosting
- Provide transparent privacy information to users
